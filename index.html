1. Two Sum

Easy

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,

return [0, 1].

Hash Table O(N)

Use a hash table to store the indexes of the elements in the array

The question asks for the indexes of the two values that sum up to target.

val1 + val2 = target

val2 = target - val1

Thus, for a given value, we can lookup the desired val2 in the hash table.

If it exists, then we return the two values indexes.
